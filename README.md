# Project Shape

### Descrição

Este projeto é um aplicativo desenvolvido com o objetivo de auxiliar no acompanhamento de uma rotina de dieta de forma simples e prática.

O foco do projeto é oferecer uma solução leve, funcional e offline, priorizando praticidade e autonomia no gerenciamento da dieta.

### Como funciona

Primeiro você define uma meta diária, incluindo peso, ingestão de calorias, macronutrientes e limite de gastos. Com isso, você pode registrar o consumo ao longo do dia, acompanhando em tempo real o progresso em relação às metas definidas.

Para incluir o consumo diario você deve realizar o cadastro de ingredientes ou receitas, os ingredientes são adicionados com seus respectivos valores nutricionais e custo, com isso, você pode criar receitas a partir desses ingredientes, calculando automaticamente informações como calorias, proteínas, carboidratos, gorduras e preço total.

O sistema organiza os dados por dia, permitindo visualizar o histórico de consumo e facilitando o controle alimentar ao longo do tempo.
